`pmac help`
===========

Display help for pmac.

* [`pmac help [COMMAND]`](#pmac-help-command)

## `pmac help [COMMAND]`

Display help for pmac.

```
USAGE
  $ pmac help [COMMAND] [-n]

ARGUMENTS
  COMMAND  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for pmac.
```
